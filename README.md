**简介**

**使用案例**

**简介**

### Universal Agent

**简介**

最小化通用 AI Agent，自动化端到端任务执行。理解自然语言意图，生成命令或脚本，执行并分析结果，自动从错误中恢复。支持三种使用模式：独立运行（需要 API Key）、桥接执行（推荐，由外部 Agent 提供 LLM）、模拟执行（最快，无设置）。
